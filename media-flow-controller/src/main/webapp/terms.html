Media Flow Controller 利用規約

本利用規約（以下「本規約」）は、Media Flow Controller（以下「本サービス」）の利用条件を定めるものです。
ユーザーは、本サービスを利用することにより、本規約に同意したものとみなされます。
本サービスは YouTube API Services を利用します。利用者は本サービスの利用により YouTube 利用規約に同意したものとみなされます。

第1条（サービス内容）
本サービスは、YouTube API Services（YouTube Data API、YouTube IFrame Player API 等）を利用し、
音楽・動画コンテンツの再生、プレイリスト生成、再生位置やスキップ区間の制御等を行う
Web アプリケーションです。

第2条（YouTube 利用規約の適用）
本サービスの利用にあたっては、本規約に加え、YouTube 利用規約
（https://www.youtube.com/t/terms）が適用されます。
ユーザーは、YouTube 利用規約を遵守するものとします。
