Media Flow Controller プライバシーポリシー

Media Flow Controller（以下「本サービス」）は、ユーザーのプライバシーを尊重し、
以下の方針に基づいて情報を取り扱います。

第1条（取得する情報）
本サービスでは、以下の情報を取得する場合があります。
・YouTube API Services を通じて取得される動画情報、プレイリスト情報等
・Google OAuth 認証により提供される認可情報
・アクセスログ（IPアドレス、ブラウザ情報等）

第2条（情報の利用目的）
取得した情報は、以下の目的のみに利用します。
・プレイリストの生成、再生制御、スキップ区間処理等の本サービス提供のため
・本サービスの動作改善および不具合対応のため

第3条（第三者提供）
本サービスは、YouTube API Services を利用しており、
Google および YouTube に対して必要な情報が送信されます。
法令に基づく場合を除き、これら以外の第三者に個人情報を提供することはありません。

第4条（情報の保存と削除）
取得した情報は、本サービスの提供に必要な範囲で保存されます。
ユーザーが Google アカウントの連携を解除した場合、
当該認可情報および関連するデータは適切な方法で削除されます。

第5条（外部サービス）
本サービスは、以下の外部 API・ライブラリを利用しています。
・YouTube Data API / YouTube IFrame Player API
・AnimeThemes API
その他、動作に必要なオープンソースライブラリ

第6条（お問い合わせ）
本ポリシーに関するお問い合わせは、以下までご連絡ください。
（例）GitHub リポジトリの Issue、または開発者連絡先

第7条（ポリシーの変更）
本プライバシーポリシーは、必要に応じて変更されることがあります。
変更後の内容は、本サービス上に掲載した時点で効力を生じます。
